# Kobek Landing Page

Welcome to the Kobek Coffee Subscription Landing Page - the official website for [kobek.sa](https://kobek.sa).

## About Kobek

Kobek is a coffee subscription service that offers fixed monthly discounts of up to 40% on your daily coffee from a carefully selected network of cafes in Saudi Arabia.

## Getting Started

> **Prerequisites:**
> The following steps require [NodeJS](https://nodejs.org/en/) to be installed on your system, so please
> install it beforehand if you haven't already.

To get started with development, you'll first need to install the dependencies with:

```bash
npm install
```

Then, you'll be able to run a development version of the project with:

```bash
npm run dev
```

After a few seconds, your project should be accessible at the address
[http://localhost:5173/](http://localhost:5173/)

## Building for Production

To build the project for production deployment on kobek.sa:

```bash
npm run build
```

This will create a `dist` folder with the optimized production build.

## Deployment to kobek.sa

The project is configured for deployment on the kobek.sa domain. The production build is optimized for:
